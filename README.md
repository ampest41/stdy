#проэкт банковского приложения
##описание
Это проэкт банковского приложения которое может:
*маскировать номера карт и счетов
*сортировать списки транзакций по ключу и времени
##установка
1. клонируйте репозиторий:
`git clone git@github.com:ampest41/stdy.git`

2. установите зависимости:
`pip install -r requirements.txt`
